# Report Template Design

Required sections:
- `Metadata`
- `Scope`
- `Current State`
- `Key Decisions`
- `Open Risks`
- `Next Actions`
- `Freshness`

Required metadata:
- `reportId`
- `reportType`
- `title`
- `sourceModules`
- `status`
- `updatedAt`

Rules:
- one report answers one bounded question
- prefer conclusions and current state over process narrative
- keep evidence references compact and structured
