# Naming Rules

Canonical rules:
- directories: `kebab-case`
- manifest ids: `dot.case`
- machine files: `snake_case.json`
- canonical docs: `Pascal-Topic.md`
- indexes: `<module>.index.md`
- templates: `<type>-template.md`
- reports: `report-<topic>-YYYYMMDD.md`
- snapshots: `snapshot-<topic>-YYYYMMDD.json`

Disallowed in canonical active files:
- vague names such as `notes.md`, `misc.md`, `temp.md`
- date-first names except report or archive artifacts
- duplicate files differing only by version suffix unless archived or explicitly roadmap-managed
